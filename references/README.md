# References

`references/` 保存项目治理相关的静态参考资料，不直接承载具体功能文档。

分类规则：

- [standards/](standards/README.md)：流程规范、角色边界、治理约束、初始化清单
- [templates/](templates/README.md)：各类标准文档模板
- [checkers/](checkers/README.md)：各类检查器的规则说明和输出契约
- [examples/](examples/README.md)：输入示例和生成报告示例

使用原则：

- 新规范放到 `standards/`
- 新模板放到 `templates/`
- 新检查规则说明放到 `checkers/`
- 新样例和样本输出放到 `examples/`
- 不要把具体功能的 PRD、设计稿、测试用例或发布记录放到 `references/`
