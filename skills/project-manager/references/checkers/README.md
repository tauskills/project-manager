# Checkers

`checkers/` 保存各类治理检查器的规则说明，而不是脚本本体。

适合放在这里的内容：

- 检查器用途和适用阶段
- 输入契约、输出契约
- 规则覆盖范围
- 风险等级和决策逻辑

不适合放在这里的内容：

- 可执行脚本
- 某个具体功能的一次性检查结果
