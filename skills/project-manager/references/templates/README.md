# Templates

`templates/` 保存标准文档模板，供具体功能文档直接复用。

适合放在这里的内容：

- PRD 模板
- UI 设计交接模板
- 架构与技术设计模板
- 测试用例模板
- 复盘模板
- 发布记录模板
- 测试报告模板
- 项目状态台账模板
