# Examples

`examples/` 保存模板输入样例和检查输出样例，用于帮助理解标准产物应该长什么样。

适合放在这里的内容：

- 示例 PRD
- 示例发布记录
- 示例检查报告

使用原则：

- 示例文件仅作为结构参考，不作为正式功能文档
- 若规范更新导致示例过时，应同步更新示例
- 示例命名应直接表达输入或输出用途
